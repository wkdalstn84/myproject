일반적인 Text File을 읽는 실습을 수행
컴퓨터에서 사용하는 File의 종류
1. Text File
 	일반적으로 윈도우의 메모장에서 열어 내용을
 	확인 할 수 있는 파일들을 말함
2. BIN(바이너리) File
 	메모장에서 열기는 할 수 있는데
 	열면 전혀 알아볼 수 없는 외계어처럼 나타나는 파일들
 	그림파일, 음악파일, 동영상파일
 	hwp문서, xls, ppt
3. FileReader는 Text File을 열어서 읽을 수 있도록
 	통로를 만들어 주는 클래스
4. BufferedReader는 FileReader가 만든 통로를 통해서
 	Text File의 Code를 가져와서 문자열로 변환시키는 클래스
5. 파일을 열기
 	모든 파일은 읽거나 내용을 쓸때(기록)
 	반드시 열과 과정을 거쳐야 한다.
 	파일을 열때는
 	
 	파일이 실제로 어디에 있는지 정확한 경로(path)를
 	지정해줘야 한다.
 	
 	com.biz.ap 패키지에 data.txt 라는 파일을 만들었는데
 	이 data.txt 파일을 java 클래스에서 열기 위해서는
 	src/com/biz/ap/data.txt 라는 경로와 파일이름을 사용한다. 	

6. 예외처리
 	프로그램코드 실행되는 중간에 exception(오류와 비슷)
 	이 발생할 수 있는데 exception 메시지는 보통 개발자를 위해서
 	보여주는 도움말이다. 하지만 실제 사용자에게는 알수없는 매우
 	불편한 존재이다. 그래서 java의 기본 exception 메시지 대신에
 	사용자가 쉽게 알 수 있도록 처리해주는 것이 좋다.
 	
 	try catch 구문이라고 한다.
